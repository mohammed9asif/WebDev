Learning the css

mainly we have three ways to apply css to html

  • 1.External css
  • 2.Internal css
  • 3.Inline css

In line css is applied to element direcly with style key attribute for eg: style = "color:red"

Internal Css applies in the head tag with style tag inside tag

External css applies through an external file and link in the head tag

Selectors for css

In internal and external css we use selectors to specify elements and apply them the styling

  1. Element selector -> using element{appyling css}
  2. class selector -> using .classname{applying css}
  3. Id selector -> using #Idname {applying css}
  4. Group selector -> selecting group of elements h1,p,li{applying css}
