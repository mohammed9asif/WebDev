This lesson is about the Position

Default position of all elements are static

static position is default , it does not move anywhere

relative postion is movable relative to parent , if we put top 100px then it will make space in top 100px means it will move downward

The imp point of relative postion is it will reserve the space of element When relatively changes , other elements wont occupy that space

  • The Other Imp position is absolute , Mainly used for overlapping
  • Similar to relative but It will not reserve the space , other elements will occupy that place
  • if any element want to put position as absolute then its parent should be marked as relative

  2. 2. fixed- stick to a postiion in the view port whole page
