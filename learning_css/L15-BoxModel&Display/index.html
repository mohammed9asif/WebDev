In this page we will be learning box model in css , display property and their values inline,block

First we will practise how to make border

border can be set by single propery border-style , by default it will be black color and some px of width

  • border-top , border-bottom,border-left , border-right are short hand for respective border
  • border radius is use to round of the edges
  • Box model has content,padding around the content, border around the padding and margin after border

    • Block Elements

  • block elementss always start from new line , if both are inline then they come in same line
  • Block elements can have width,heigght,padding ,margin

    • In Line elements

  • Inline elements takes only space of the content, if space is present it will start in same line
  • inline elements cant have height, width ,margin top and bottom
  • they can have only padding ,margin left,right

    • Inline-Block elements

  • Inline-Block is specialized property where it has both good parts of the block and Inline elements
